The sage accepts the World
As the World accepts Tao;
He does not display himself,
so is clearly seen,
Does not justify himself,
so is famed,
Does not boast,
so is credited,
Does not glory,
so excels,
Does not contend,
so no one contends against him.

— Lao Tzu, Tao Te Ching
